# Webtechpie BE
Official backend for webtechpie.com

## Development

### Prerequisites

- Node.js (>= 22.14.0)
- Yarn
- VS Code

### Installation

```bash
git clone https://github.com/webtechpie-be/backend.git
yarn install
```

Run app in dev mode

```
yarn start:dev
```

## Build

```
yarn build
```
